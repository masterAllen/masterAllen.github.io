TCL Q10G Pro 电视解锁ADB

2023.04.06

1. 解锁ADB

设置→关于本机→本机信息，依次按遥控器的上下左右，就会出现adb选项，将其打开。

设置→网络→有线/wifi(选择你使用的网络)→网络详情，查看电视的ip，例如我这边是192.168.1.12。

在电脑上安装adb，使用adb connect <ip> (例如adb connect 192.168.1.12 )命令进行连接，电视上会出现授权请求，授权之后就可以远程连接adb了。

2. 解除安装限制

网传解除adb安装限制的命令为在 adb shell 中执行setprop persist.tcl.installapk.enable 1，但是经过测试并没有效果，使用adb install命令安装apk时，报错为[INSTALL_FAILED_VERIFICATION_FAILURE]。

经过对报错的搜索，发现执行这两条命令即可解除(来源: https://stackoverflow.com/questions/15014519)

settings put global verifier_verify_adb_installs 0
settings put global package_verifier_enable 0

3. 其他

TCL的新版桌面可以完全自定义，目前我只留了kodi、天气在桌面上，非常清爽。搭配mosdns，在DNS层面进行广告过滤，至今我没有在电视上看到任何广告，有需要的朋友可以尝试一下。
