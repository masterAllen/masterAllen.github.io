    Linux 处理 PDF 常用命令

    1. 获取文件的某些页
      # 相当于去掉封面
pdftk in.pdf cat 2-end output out.pdf

    2. 合并文件

      pdftk in1.pdf in2.pdf in3.pdf cat output out.pdf

    3. 添加页码

      # 底部，大小 16
cpdf -bottom 40 -font Arial -font-size 16 -add-text "%Page" in.pdf -o out.pdf
